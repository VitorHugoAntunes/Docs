# Começando com Estruturas de Dados

Este guia introduz os fundamentos de **estruturas de dados** em **C** e **Java**, ideal para iniciantes e estudantes que buscam uma base sólida.

## Por que estudar estruturas de dados?

Estruturas de dados organizam informações de forma eficiente, sendo essenciais para criar programas otimizados. Neste guia, você vai:
- Compreender o funcionamento de diferentes estruturas.
- Implementar exemplos em C e Java.
- Aplicar conceitos em problemas práticos.

## Pré-requisitos

**Para C**:

Compiladores:
  - [GCC](https://gcc.gnu.org/) – GNU Compiler Collection
  - [MSVC](https://learn.microsoft.com/pt-br/cpp/windows/latest-supported-vc-redist?view=msvc-170) – Microsoft Visual C++ (via Visual Studio)
  - [MinGW](https://www.mingw-w64.org/) – Minimalist GNU for Windows

Editores/IDEs:
  - [Code::Blocks](http://www.codeblocks.org/)
  - [Dev-C++](https://www.bloodshed.net/)
  - [Visual Studio Code](https://code.visualstudio.com/)

Alternativas online:
- [Replit C](https://replit.com/languages/c)

**Para Java**:
- JDK 17 ou superior ([documentação oficial](https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html)).
- Editor ou IDE (ex.: [IntelliJ](https://www.jetbrains.com/idea/), [Eclipse](https://eclipseide.org/)).
- Alternativa online: [Replit Java](https://replit.com/languages/java10).

**Conhecimentos**:
- Noções básicas de programação (variáveis, funções, loops).
- Entrada/saída em C (printf/scanf) ou Java (System.out.println/Scanner).

## Estrutura do Guia

Cada seção cobre:
1. Conceitos teóricos claros.
2. Códigos de exemplo em C e Java.
3. Exercícios práticos para aprendizado.
